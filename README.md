## 🚀 Features

- Auto-detects columns (sales, category, region, dates)
- Executive Dashboard with KPIs
- Sales Forecasting
- Seasonality Analysis
- Heatmaps (Category vs Region)
- AI Agent – ask questions in plain English
- Upload your own CSV – works with any data
